# E-commerce Platform

## Overview
The e-commerce-platform is a comprehensive online shopping platform designed to provide a seamless and intuitive user experience.

## Setup
1. Install Docker
2. Clone the repository
3. Create a .env file with the required environment variables
4. Build the Docker image: `docker build -t ecommerce .`
5. Run the Docker container: `docker run -p 8000:8000 ecommerce`
